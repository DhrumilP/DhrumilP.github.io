/PROJECTS

 
 
 
 
 

BioJava

Java Java Swing

An application developed using Java and Java Swing to simulate famous biology convention called Punnet square. This software will calculate and output the chances for offspring having certain traits when information for parents are given. The software is also accompanied by basic definition module. This application is also accompanied with a quiz module.

Autonomous Car

Arduino Sensors

The Arduino self-driven car assembled using stepper and servo motor, IR receiver, remote control, ultrasonic sensor, Capacitors and LCD screen. This project was developed for Institute of Robotics and Intelligence Systems and was implemented as one of the workshops that this organization offers.

 
 
 
 
 
 
 
 
 
 

Food Print

Python Open CV MongoDB

Web application developed during Deltahacks VI. Users are able to capture or upload a picture of a receipt from a grocery store and the carbon footprint of their groceries will be calculated. It is made using OpenCv for Optical Character Recognition of the receipts and then using algorithm written in python it extracts the information and cross refrences it with the database to output the carbon footprint.

Personal Website

HTML Bootstrap 4 Particle.js

Personal website created using HTML, CSS, JavaScript and Bootstrap. This website is a digital resume which includes some of my projects and skills. Currently on the first version and planning to incorporate various other frameworks too to make it more interactive and other components as well.
